1 Open Hubitat Web Interface
Navigate to your Hubitat hub at:
http://YOUR-HUB-IP-ADDRESS
Example: http://192.168.1.100 or http://hubitat.local
2 Find Maker API
  1. Click "Apps" in the left sidebar
  2. Click on "Maker API" in your installed apps
  3. If you don't see it, click "Add Built-In App" ‚Üí Select "Maker API"
3 Copy Your Cloud Endpoint URL
Inside the Maker API app, scroll down to find:
"Get All Devices with Full Details"

You'll see a URL like:
https://cloud.hubitat.com/api/abc-123/apps/12/devices?access_token=xyz789
